You also need to declare a new variable i for each new sec to find out later the last common index between these two words res and sec.

I still need a hint.

You need to increment this i until it's smaller the minimum of current value of maximum index l and length of the sec element and until characters at this index at both elements (res, sec) are the same. Let's write it. 

while (i < Math.min(l, sec.length) && res[i] === sec[i]) i++;
How to finish the code?

It's almost end. Now you need to update your stored maximum index of common part with a new value of i. 

l = i

Here, you can make immediate checking. If l == 0 it means no common part left already and you may return empty string without further popping. 

if (!(l = i)) return "";
So, how my final solution should look like?

You just need to join all previous hints, return a slice of the first element till final common index and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

function longestPrefix(arr: string[]): string {

    let res = arr.pop(), l = res.length;
    while (arr.length) {
        let sec = arr.pop(), i = 0;
        while (i < Math.min(l, sec.length) && res[i] === sec[i]) i++;
        if (!(l = i)) return "";
    }
    return res.slice(0, l);
}

Using reduce function to compare each word with the base and keep only common part as base word. 

function longestPrefix(arr: string[]): string {

    if (!arr.length) return "";
    return arr.reduce((prev, next) => {
        let i = 0;
        while (i < prev.length && i < next.length && prev[i] === next[i]) i++;
        return prev.substring(0, i);
    });
}
